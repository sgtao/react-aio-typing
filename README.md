# react-aio-typing

## 概要

- 書籍『ALL IN ONE』の英文を使ったブラウザ型タイピング学習ゲーム
- **タイピング練習**（英文を見て入力）と**英作文**（日本語訳から英文を入力）の2モードを備える
- フルスペックのソフトは、Linkage Club『[ALL IN ONE](https://linkage-club.net/books#all)』の書籍を参考にしてみてください
  - ALL IN ONE『[英語独学支援サイト](https://www.aio-english.net/)』

## 操作方法

### 01．ホーム画面

設定パネルでモードと動作を選択してからスタートする。

#### モード選択

| モード | 内容 |
|--------|------|
| **タイピング練習** | 英文を見ながらそのまま入力する |
| **英作文** | 日本語訳（＋ヒント）を見て英文を自力で入力する |

#### 設定項目

| 設定 | 選択肢 | 説明 |
|------|--------|------|
| ミスタイプ | ストリクト / フリー | ストリクト: 誤キーで入力がブロックされる。フリー: そのまま進み Backspace で修正できる |
| 日本語訳 | スラッシュ訳 / 自然な訳 | タイピング中に表示する訳のスタイル |
| ヒントレベル（英作文のみ） | Lv.1 易 / Lv.2 中 / Lv.3 難 | ヒントエリアの表示量を制御する |

| キー | 機能 |
|------|------|
| Enter / Click | スタート |

### 02．タイピング中

| キー | 機能 | 備考 |
|------|------|------|
| `0-9a-zA-Z` | タイピング入力 | 句読点・スペースは自動スキップ |
| Backspace | 1文字戻す | フリーモードのみ |
| Enter | 音声の再生 / 一時停止 | |
| Tab | 訳スタイル切り替え | スラッシュ訳 ↔ 自然な訳 |
| Escape | 現在の文をリセット・音声を停止 | |

#### ライブスコア

タイピング中に **WPM・正確性・経過時間** をリアルタイム表示する。

#### 英作文モードのヒントレベル

| レベル | ヒントエリア表示 | 例 |
|--------|----------------|----|
| Lv.1 易 | 英文そのまま | `She is a teacher.` |
| Lv.2 中 | 各単語の先頭文字のみ | `S__ i_ a t______.` |
| Lv.3 難 | ヒントなし | （空欄） |

### 03．結果画面

1文タイプ完了後に表示される。

| 表示項目 | 内容 |
|---------|------|
| **WPM** | 1分あたりのタイプ語数（Words Per Minute） |
| **正確性** | 正しく入力できた割合（%） |
| **タイム** | 経過時間 |
| **ハイライト** | 正解・ミス箇所の色分け表示 |

| キー | 機能 |
|------|------|
| Enter | 次の文へ |
| Escape | メニューへ戻る |
